Recomendaciones del juego: En el comienzo del juego se tiene ya construida una fuente. Esta al presionarse sube el 
nivel de agua. 
En el menú de construcciones está todo lo que quisieras construir. Para construir una granja (cocoa o wheat) debes primero
construir una farmland para plantarla.
El barn aumenta la capacidad de almacenamiento de productos

Hay Varios tipos de edificios productores, la cocoa, el wheat, el eggs factory, el milk factory, el mill y el bakery. Todos siempre y cuando estén 
construidos, generan recursos, no obstante se le debe asignar la producción. Para hacerlo deja presionado el edificio y el boton Funct te dará las opciones de producción del edificio. Después de una cantidad de tiempo que se pueden recoger. Se sabe porque encima
del edificio se verá un spriteo. Para recolectar su producción se le da click
izquierdo

Si se desea reubicar un edificio se puede dejar presionado y al abrirse dos botones, presionar M de mover. El D es para destruirlo

En el mercado puedes donarle al pueblo para aumentar su nivel de supplies en la store y puedes vender para sobrevivir. Vende cuanto quieras de cada producto usando el slider de cada item en la tienda. Usando los botones inferiores puedes seleccionar el modo que prefieras de comercio, donar, vender o comprar. (comprar no sirve aún)
 
En el juego se pierde cuando los animales mueren de hambre. El Time.Scale se detiene y se hace print "Game Over"

En el juego se gana cuando pasado un tiempo la gente aún tiene supplies. Pasaste el día.
